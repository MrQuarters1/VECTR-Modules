## Learning Objectives
- Understand the purpose of SSL/TLS in securing network communication.
- Identify how encryption protects data in transit.
- Recognize the role of certificates in authentication.
- Demonstrate basic usage of HTTPS.

## Prerequisites
- Kali Linux VM
- Basic networking knowledge
- Web browser

## Lab Environment Setup

Kali Base:

RUN sudo apt-get update -y  
RUN sudo apt-get install curl -y  

## Assessment

### Assessment 1:
- Use curl to connect to an HTTPS website
- Observe the secure connection

### Assessment 2:
- Compare HTTP vs HTTPS in a browser
- Identify the security difference

### Assessment 3:
- Explain what SSL/TLS does in securing communication

### Assessment 4:
- Identify where certificates are used in HTTPS

### Assessment 5:
- Describe why encryption is important for network security
