## Learning Objectives
- Use **Wireshark** to capture packets.
- Identify different protocols within the packet data.
- Filter packets to find one particular source address.

## Prerequisites
- none


## Lab Environment Setup
Kali Base
```bash
RUN sudo apt-get install wireshark -y
RUN sudo apt-get install iputils-ping -y
RUN sudo apt-get install traceroute -y
RUN sudo apt-get install dnsutils -y
RUN sudo apt-get install nmap -y
```

## Assessment
### Assessment 1:

- Save .pcap file
